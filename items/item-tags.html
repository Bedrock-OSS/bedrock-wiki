Table of Contents
Item Tags
    1. Applying Tags
    2. Testing for Tags
    3. Lists of Vanilla Item Tags

Item tags can be used to ensure that a item meets certain conditions.

Applying Tags #

From 1.20.50 and onwards #

Copy
json
{
	"format_version": "1.20.50",
	"minecraft:item": {
		"description": {
			"identifier": "example:my_item"
		},
		"components": {
			"minecraft:tags": {
        "tags": [
          "example:my_tag"
        ]
      }
		}
	}
}

Testing for Tags #

Tags can be queried with:

  • q.all_tags
  • q.any_tag
  • q.equipped_item_all_tags
  • q.equipped_item_any_tag
